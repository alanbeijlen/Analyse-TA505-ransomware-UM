# Ransomware Threat Analysis & Risk Assessment 🛡️

## Over het Project
In dit onderzoek heb ik een diepgaande analyse uitgevoerd op een professionele ransomware-actor, toegepast op een MKB-telecomfranchise. Het doel was om zowel de technische aanvalspaden als de financiële en operationele risico's in kaart te brengen en om te oefenen met het gebruik van GitHub.

## Kernonderdelen
* **Dreigingsanalyse:** Mapping van tactieken en technieken op de **Cyber Kill Chain**, **Unified Kill Chain** en **MITRE ATT&CK**
* **Kwantitatieve Risicoanalyse:** Gebruik van de **RAVIB-methodiek** om investeringsadvies te onderbouwen met concrete metrics zoals **SLE**, **ALE** en **ROSI**
* **Compliance:** Getoetst tegen de **ISO 27001:2022** audit-eisen (93 beheersmaatregelen)
* **Incident Response:** Forensisch onderzoeksplan opgesteld conform **ISO/IEC 27037** en het **NCSC First Responder-protocol**

## Gebruikte Methodieken & Frameworks
| Categorie | Tools / Frameworks |
| :--- | :--- |
| **Frameworks** | MITRE ATT&CK, Cyber Kill Chain, PTES  |
| **Compliance** | ISO 27001:2022, NIS2, AVG/GDPR  |
| **Risico** | RAVIB, BIV-classificatie  |

## Conclusie
Het resultaat is een strategisch verbeterplan gericht op het verhogen van het **SANS-volwassenheidsniveau**, inclusief een juridisch kader voor implementatie.
